# webpack-pug

`$ yarn init`  
`$ yarn add -D webpack webpack-cli webpack-dev-server`

由於需要輸出HTML，所以需要安裝html-webpack-plugin

`$ yarn add -D html-webpack-plugin`  

`$ yarn add -D pug html-loader pug-html-loader`  

`$ yarn add -D gh-pages`
